# 🎓 Student CRUD Full Stack Project

A simple full-stack **Student Management System** built using:

- Backend: Spring Boot (Java)
- Frontend: React (Vite)
- Database: H2 (in-memory/file-based)

This project demonstrates full CRUD operations with REST APIs.

---

## 🚀 Features

- Add Student
- View Students
- Update Student
- Delete Student
- Form Validation
- Custom Popup Notifications
- REST API integration

---

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 Database
- Maven

### Frontend
- React
- Vite
- Axios
- JavaScript
- CSS

## ⚙️ How to Run

### Backend (Spring Boot)

```bash
cd backend
./mvnw spring-boot:run

Runs at:
http://localhost:8080

Frontend (React)
cd frontend
npm install
npm run dev

Runs at:
http://localhost:5173


🔌 API Endpoints
Method	Endpoint	Description
GET	/students	Get all students
GET	/students/{id}	Get student by ID
POST	/students	Create student
PUT	/students/{id}	Update student
DELETE	/students/{id}	Delete student
